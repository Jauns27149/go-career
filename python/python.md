# pip

-  pip install <package-name> : 安装包
- pip install --upgrade <package-name> : 升级包

# 关键字

| 关键字 | 作用                         |
| ------ | ---------------------------- |
| as     | 用于给对象或异常指定一个别名 |
| raise  | 手动引发（抛出）异常         |
| and    | 逻辑运算符and                |
| pass   | 占位语句                     |



# 符号

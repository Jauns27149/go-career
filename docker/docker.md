# 命令

| 命令                        | 作用                             |
| --------------------------- | -------------------------------- |
| docker ps                   | 列出当前正在运行的容器           |
| docker images               | 列出本地机器上的所有 Docker 镜像 |
| docker rm                   | 删除一个或多个已停止的容器       |
| docker rmi                  | 删除一个或多个镜像               |
| docker build  -t name:tag . | 利用Dockerfile文件打包镜像       |
